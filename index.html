METADATA OBJECT
This object is returned when the id3 data has been parsed. It returns a series of relative timestamps in seconds.
  data: {
    id: demuxer id,
    frag : fragment object,
    samples : [
      id3 pes - pts and dts timestamps
      are relative, values are in seconds
    ]
  }
